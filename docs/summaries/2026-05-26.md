## Important changes

- **Documentation is now a first-class deliverable.** Every PR going forward should add a DECISIONS.md entry if it's not a trivial copy tweak, and roll up into a daily summary at end-of-session.
- **DECISIONS.md is append-only.** Reversed decisions stay in place with a new "Supersedes 0XX" entry — never edit history. This matters because future-me will ask "why didn't they just do the simpler thing?" and the answer is in the prior entry.
- **Summary file naming convention is literal `YYYY-MM-DD.md`** so `ls docs/summaries/` sorts chronologically without extra tooling.
